Quick Install

Install the latest version of CloudOS with a single command:

curl -sSL https://install.cloudos.dev | bash

Custom Installation

Customize your installation with environment variables:

CLOUDOS_INSTALL_DIR=$HOME/.local/cloudos \
CLOUDOS_VERSION=v1.0.0 \
curl -sSL https://install.cloudos.dev | bash

Options

  • CLOUDOS_VERSION - Specific version to install (default: latest)
  • CLOUDOS_INSTALL_DIR - Installation directory (default: /opt/cloudos)
  • CLOUDOS_ARCH - Force architecture (x86_64, arm64, armv7)
  • FORCE_INSTALL - Force reinstall (default: false)
